## Project Overview
This project is a comprehensive web application that serves as a walkthrough and guide for the video game Stellar Blade. The goal is to provide detailed information about the game, including collectibles, main story walkthrough, side quests, characters, and items. The guide is designed to be user-friendly and interactive, allowing users to comment and engage in discussions about various aspects of the game.

### Current Status
This project is in active development, with the Collectibles Guide, user accounts, and commenting/discussion features fully implemented. Core features are functional, and further design improvements are in progress. Regular updates are being committed as development progresses.

## Features
### Technical Features
The project utilizes the following libraries and features:
- **MUI:** The `@mui/material` library provides skeleton loaders for a consistent visual design while content loads.
- **GLightbox:** Used for image gallery and lightbox functionality, enhancing the presentation of collectibles.
- **Local Storage:** Caches data locally, reducing load times and redundant API requests for improved performance.
- **Lazy Loading and Suspense:** React's `lazy` and `Suspense` features implement code splitting and lazy loading, optimizing load times.

### Completed Features
- **Collectibles Guide:** Detailed sections on all in-game collectibles, with descriptions and screenshots.
- **Image Gallery with GLightbox:** Enhanced image gallery with a lightbox format and smooth transitions.
- **Mobile Optimization:** Responsive design optimized for mobile devices.
- **Local Storage API Caching:** Reduces load times by minimizing redundant API calls.
- **User Accounts and Authentication:** Allows users to register and log in to personalize their experience.
- **Commenting and Discussion System:** Secure discussion system with rate limiting and input sanitization for safe, organized conversations.

### Planned Features
- **Main Story Walkthrough:** Step-by-step guide through the main story.
- **Side Quests:** Information and guidance on all side quests.
- **Character and Item Information:** Detailed descriptions and stats.
